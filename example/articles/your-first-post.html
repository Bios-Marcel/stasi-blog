This is a demo blog post, it aims to show you what stasi-blog can do and how to get started.

Feature Overview

  • Article overview with paging and tags
  • RSS Feed
  • Mobile friendly
  • Automatic Darkmode / Lightmode
  • Custom Pages (Example would be an About page)
  • Fast to load even with a slow (less than 64kbit/s) internet connection

Desktop-only Features

  • Optional `Tags` sidebar for filtering by category

Features reliant on JavaScript

  • Comments via utteranc.es (via GitHub issues)
